
#### pip更新
```pip
pip install --upgrade pip
```

#### pip显示特定库信息
```pip
pip show <library_name>
```

#### 临时使用其他源
```pip
pip install <package> -i <url>
```
清华源url：https://pypi.tuna.tsinghua.edu.cn/simple

#### 搜索指定库
```pip
pip search <package>
```
PyPI已不支持此命令 Please use https://pypi.org/search (via a browser)
